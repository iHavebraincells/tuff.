General
Court.larp is a collection of free Discord bots built to give server owners and moderators powerful tools. We currently offer two bots — Raid and Nuke — each focused on different aspects of discord.

Setup
Click the "Add Raid" or "Add Nuke" button on the homepage. You'll be taken to Discord's authorization page where you can choose which server to add the bot to. You'll need the "Manage Server" permission on that server.
Court.larp Raid does not need any permission you simply add it to your account to use it in any server. Court.larp Nuke requests administrator permissions since it performs a server nuke action. You can review exactly what permissions are requested on Discord's authorization screen before confirming.

The Bots
Court.larp Raid is focused on spamming messages in a channel raiding it. Court.larp Nuke is designed for more advanced, high-impact server damage that bans members, wipes all channel and roles, pings everyone 9,000+ times, that require administrator-level permissions. If you're unsure, start with Raid.
If a bot is not responding, first check that it has the correct permissions in your channel. If the bot appears offline, please report it in our Discord server and we'll investigate immediately.
We focus on keeping Raid and Nuke high-quality rather than shipping lots of bots. That said, community suggestions do influence the roadmap. Join the Discord and let us know what you'd like to see.
